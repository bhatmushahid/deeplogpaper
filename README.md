# DeepLog - Heba's Branch

## Scripts
- `parse_hdfs.py` — Parses raw HDFS.log using Drain3, outputs parsed_hdfs.csv
- `extract_parameters.py` — Builds log key vocabulary, extracts parameters, outputs structured_hdfs.csv

## Setup
```bash
source ~/hdfs_env/bin/activate
cd ~/DeepLog
```

## Data
Dataset not included (too large). Download HDFS_v1 from LogHub GitHub.
